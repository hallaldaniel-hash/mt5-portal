Existing group import wizard

            
Use this when an MT5 account already had activity before the portal existed.

          

        

        

          
What this does
It converts old MT5 cash movements into clean ledger explanations.

          
Important
This step is manual for now. It does not scan live MT5 yet. You add movements like +500 deposit or -700 withdrawal, then classify what they really were.

          
Safe flow
Queue movement → Review → Finalize. Nothing is saved to the ledger until Finalize.
